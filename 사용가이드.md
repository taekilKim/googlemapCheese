# 📍 구글 지도 인증샷 메이커 - 사용 가이드

## ✨ 주요 개선 사항

1. **🔥 구글 지도 링크 자동 연동**: 링크만 붙여넣으면 장소 정보 자동으로 가져오기!
2. **📸 사진 잘림 개선**: 카드가 사진에 살짝만 겹쳐서 사진이 훨씬 더 많이 보임

---

## 🚀 빠른 시작 (5분 소요)

### 1단계: Google Maps API 키 발급 (최초 1회만)

#### 방법 A: 상세 가이드
1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 프로젝트 생성:
   - 상단에 "프로젝트 선택" 클릭
   - "새 프로젝트" 클릭
   - 프로젝트 이름 입력 (예: "지도인증샷")
   - "만들기" 클릭

3. API 활성화:
   - 좌측 메뉴에서 "API 및 서비스" → "라이브러리" 클릭
   - "Places API" 검색
   - "Places API" 클릭 → "사용" 버튼 클릭

4. API 키 생성:
   - 좌측 메뉴에서 "API 및 서비스" → "사용자 인증 정보" 클릭
   - 상단에 "+ 사용자 인증 정보 만들기" → "API 키" 클릭
   - API 키가 생성됨! (예: AIzaSyC2Xj...)
   - 복사해두기

5. (선택) API 키 제한 설정:
   - API 키 옆의 편집(연필) 아이콘 클릭
   - "애플리케이션 제한사항": "HTTP 리퍼러(웹사이트)" 선택
   - "웹사이트 제한사항": `file://*` 추가 (로컬 파일에서 사용하려면)
   - "API 제한사항": "키 제한" 선택 → "Places API" 체크
   - "저장" 클릭

#### 방법 B: 간단 링크
👉 [직접 API 키 생성 페이지로 이동](https://console.cloud.google.com/google/maps-apis/credentials)

### 2단계: 앱 사용하기

1. **`maps-photo-final.html`** 파일을 브라우저에서 열기

2. **API 키 입력**
   - 발급받은 API 키를 복사해서 붙여넣기
   - 자동으로 저장됨 (다음부터는 입력 불필요)

3. **구글 지도 링크 복사**
   ```
   구글 지도 앱/웹 → 장소 검색 → 공유 버튼 → 링크 복사
   예: https://maps.app.goo.gl/xxxxx
   또는: https://www.google.com/maps/place/...
   ```

4. **정보 가져오기**
   - "🔍 장소 정보 자동으로 가져오기" 버튼 클릭
   - 장소명, 평점, 주소 등이 자동으로 입력됨!

5. **사진 업로드**
   - 방문했던 장소 사진 업로드

6. **이미지 저장**
   - "💾 이미지 저장" 클릭
   - 인스타그램/카카오톡에 공유!

## 🛠️ 문제 해결

### Q: "API 키가 유효하지 않습니다" 오류
**A:** 
1. Places API가 활성화되었는지 확인
2. API 키를 올바르게 복사했는지 확인
3. 결제 정보를 등록했는지 확인 (무료 크레딧 사용을 위해 필요)

### Q: "REQUEST_DENIED" 오류
**A:**
1. Google Cloud Console → "API 및 서비스" → "사용 설정된 API"
2. "Places API"가 목록에 있는지 확인
3. 없다면 라이브러리에서 활성화

### Q: 링크를 붙여넣었는데 정보가 안 나와요
**A:**
- 구글 지도에서 **장소를 선택한 후**의 링크를 사용해야 합니다
- 단순 검색 URL이 아닌, 특정 장소의 링크여야 합니다
- 올바른 예: `https://maps.app.goo.gl/xxxxx`
- 올바른 예: `https://www.google.com/maps/place/...`

### Q: 사진이 여전히 많이 잘려요
**A:**
- 세로로 긴 사진보다는 정사각형에 가까운 사진 사용 권장
- 중요한 부분이 위쪽에 오도록 구도 조정
- 카드 높이는 구글 지도 실제 UI에 맞춰져 있습니다

### Q: CORS 오류가 발생해요
**A:**
- 현재 allorigins.win 프록시를 사용 중입니다
- 프록시가 느리거나 작동하지 않으면 다른 프록시로 변경:
  - `https://corsproxy.io/?${apiUrl}`
  - `https://cors-anywhere.herokuapp.com/${apiUrl}`

## 🎯 팁 & 트릭

### 더 예쁜 인증샷 만들기
1. **조명**: 밝은 낮에 촬영한 사진이 가장 예쁨
2. **구도**: 장소의 특징이 잘 드러나는 각도
3. **편집**: 사진 밝기/대비를 살짝 조정한 후 업로드
4. **일관성**: 같은 필터를 사용하면 피드가 통일감 있음

### API 키 관리
- API 키는 브라우저의 localStorage에 저장됨
- 다른 기기에서 사용하려면 다시 입력 필요
- 보안을 위해 정기적으로 API 키 재생성 권장

---

## 🔐 개인정보 보호

- API 키는 **브라우저에만 저장**됩니다
- **서버로 전송되지 않습니다**
- 모든 처리는 **사용자의 브라우저에서만** 이루어집니다
- 사진은 기기에만 저장되며 업로드되지 않습니다
